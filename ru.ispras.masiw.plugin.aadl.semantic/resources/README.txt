In current state "problems.xml" file in this folder is not used. The file which is used is located in 
"ru.ispras.masiw.aadl/resources". File in this folder is partial copy of it.

Folder TestCorpus contains test AADL specifications for different semantic mistakes. Each specification
name allows to find place, where corresponding AADL rule can be found in standard AS5506b.

Approximate state of current checked rules can be found in spreadsheet:
https://docs.google.com/spreadsheets/d/1DXR5qEenXme6-_o2-JMuqrVXwANz3Oi9yNP7uppAGQg

Though, current version of semantic checker is a prototype and will be modified later.
